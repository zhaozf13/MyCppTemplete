# MyCppTemplete
用于记录数据结构学习中算法的C++实现，主要学习的课程来自牛客网左程云的算法课程。
该课程所有算法是Java编写的，故做了C++版本的整理和改写。
在整理过程中也夹杂了一些经典问题的解法。
链接：https://www.nowcoder.com/courses/semester/2020algorithm-base
目前整理的算法包括
1.位运算实现加减乘除
2.二叉树的实现和遍历
3.位图
4.监狱关押囚犯问题
5.图的实现
6.KMP算法
7.单向链表的实现
8.Manacher算法
9.Morris遍历
10.统计数组中数字出现次数，时间复杂度o(N),空间复杂度o(1)
11.快速排序——进阶版
12.逆序一个栈
13.并查集
14.背包问题
15.汉诺塔问题
16.动态规划之找零钱问题
